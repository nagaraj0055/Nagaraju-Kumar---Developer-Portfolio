## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Dark Glassmorphism UI** | Sleek dark theme with frosted-glass cards, soft glows, and gradient accents |
| ⚡ **Ultra-Smooth Scrolling** | Custom `requestAnimationFrame` smooth scroll with eased cubic interpolation |
| 🔄 **Scroll-Triggered Animations** | Fade-in, slide-in, and scale animations powered by Intersection Observer API |
| ⌨️ **Typing Effect** | Auto-typing hero text cycling through roles (Software Developer, Backend Enthusiast, etc.) |
| 🖱️ **Cursor Glow** | Radial gradient follows mouse movement on desktop for a premium feel |
| 🃏 **3D Tilt Cards** | Project cards respond to mouse position with perspective transforms |
| 📊 **Animated Skill Bars** | Skill proficiency bars animate on scroll into viewport |
| 🔢 **Counter Animation** | Stats count up from 0 with eased cubic timing |
| 🌌 **Floating Particles** | Randomized particle system in the hero background |
| 📱 **Fully Responsive** | Optimized layouts for desktop, tablet, and mobile devices |
| 🍔 **Mobile Navigation** | Animated hamburger menu with slide-in panel |
| ✨ **Micro-Interactions** | Button ripple effects, hover transforms, and animated form inputs |
| ⏳ **Preloader** | Triple-ring loading animation with branding |
| 🎯 **Active Section Tracking** | Navigation highlights current section on scroll |

## 🚀 Getting Started

### Prerequisites
No build tools, package managers, or dependencies required — just a browser.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nagaraju-0055/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open the file
   open index.html        # macOS
   start index.html       # Windows
   xdg-open index.html    # Linux
   ```

3. **Or use a local server** (recommended for best experience)
   ```bash
   # Using Python
   python -m http.server 8888

   # Using Node.js
   npx http-server . -p 8888
   ```
   Then visit `http://localhost:8888`

## 🎨 Design Highlights

- **Color Palette:** Deep navy (`#0a0a0f`) with purple-cyan gradient accents (`#6c63ff` → `#a855f7` → `#06b6d4`)
- **Typography:** [Inter](https://fonts.google.com/specimen/Inter) (body) + [Poppins](https://fonts.google.com/specimen/Poppins) (headings)
- **Effects:** Glassmorphism (`backdrop-filter: blur`), neumorphic shadows, CSS-only scroll indicator
- **Animations:** All animations use `cubic-bezier` easing for premium, fluid motion

---

## 🧰 Technologies Used

- **HTML5** — Semantic structure, accessibility attributes, SEO meta tags
- **CSS3** — Custom properties, Grid, Flexbox, `clamp()`, `@keyframes`, `backdrop-filter`
- **JavaScript (ES6+)** — Intersection Observer, `requestAnimationFrame`, event delegation, DOM manipulation

> **Zero dependencies.** No React. No Angular. No Bootstrap. No Tailwind. Just clean, hand-crafted code.
